How It Works

Automated tracking process that handles everything from Excel upload to data extraction and export

1
Upload Excel File

Upload your Excel file containing BL numbers and carrier names. Our system automatically reads and processes the data.

  • Supports .xlsx and .xls formats
  • Automatic column detection
  • Data validation and error checking
  • Secure file processing
2
Data Processing & Categorization

The application reads the Excel file and intelligently categorizes data by shipping line for efficient processing.

  • Groups BL numbers by carrier
  • Validates carrier compatibility
  • Creates optimized processing queues
  • Handles duplicate entries
3
Automated Web Scraping

Our system initializes Chrome browser with Selenium and navigates to each carrier's tracking page automatically.

  • Automated browser initialization
  • Carrier-specific navigation logic
  • Intelligent waiting for page loads
  • Captcha handling when required
Selenium Chrome Driver Python Automation
4
Real-time Data Extraction

For each BL number, the system searches the carrier's tracking page and extracts shipment status, location, and ETA.

  • Sequential BL number processing
  • Real-time status extraction
  • Transit time calculation
  • Error handling and retry logic
5
Batch Processing by Carrier

Complete all BL numbers for one shipping line before moving to the next, ensuring efficient resource usage.

  • Processes all BLs per carrier in batch
  • Cleans browser session between carriers
  • Maintains data integrity
  • Progress tracking per carrier
6
Data Compilation & Export

After completing all shipping lines, the application compiles all extracted data and saves it back to Excel format.

  • Consolidates all extracted data
  • Calculates transit times and ETAs
  • Formats data for Excel export
  • Creates detailed tracking report
